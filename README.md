# SomeThingFunning
综合技术应用，把会的都搞一搞

---------------实现功能------------------
(1)	用户注册
(2)	用户登录
(3)	用户权限分级管理
(4)	根据权限开放对应功能模块
(5)	用户可以发帖灌水
(6)	可以上传下载文件
(7)	可以在线观看用户上传视频、图片
(8)	用户可以发布需要付款的视频、图片
(9)	有专门的后台管理界面
(10)	模块化功能配置
---------------整体风格------------------
首页类似于公司OA，可以看到对应的信息
个人信息管理
